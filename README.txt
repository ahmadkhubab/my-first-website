
My First Website

Welcome to My First Website! This project is a simple HTML and CSS website with basic JavaScript functionality. It includes several key pages: Home, About Us, Contact Us, Sign Up, and Sign In.

Project Structure

The project structure is as follows:

- index.html: The homepage of the website.
- aboutus.html: The About Us page, describing the purpose of the website.
- contactus.html: The Contact Us page, containing contact information.
- signup.html: The Sign-Up page for new users to create an account.
- signin.html: The Sign-In page for returning users to log in.
- css/style.css: The stylesheet containing the CSS styles for the website.
- js/pass_validate.js: The JavaScript file for form validation.

Detailed File Descriptions

index.html
The homepage of the website includes:
- Navigation links to the Sign Up, Sign In, About Us, and Contact Us pages.
- A logo image.
- Introductory text about learning HTML and CSS.

aboutus.html
The About Us page includes:
- Navigation links similar to the homepage.
- Information about the purpose of the website and the technologies used.

contactus.html
The Contact Us page includes:
- Navigation links similar to the homepage.
- Contact details including a phone number and email address.

signup.html
The Sign-Up page includes:
- A form for users to create a new account.
- Fields for Name, Email, Password, and Confirm Password.
- Password validation using JavaScript to ensure at least one uppercase letter and one special character.

signin.html
The Sign-In page includes:
- A form for users to log in.
- Fields for Name, Email, and Password.

css/style.css
The stylesheet for the website includes:
- General body styling.
- Specific styles for elements such as borders, fonts, and colors.
- Styles for list elements.

js/pass_validate.js
The JavaScript file includes:
- A function to validate that the passwords match.
- A function to check that the password contains at least one uppercase letter and one special character.

3. Open index.html in your web browser:
   You can open it directly from your file explorer or use a local server if preferred.
